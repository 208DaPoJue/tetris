# **tetris**

**a web page tetris**

## run method
*when you first run, make sure you have install dependencies(run web/setup.sh)*

1. cd web

2. gulp

   > will create a dist document

3. cd ../dist

   python3 server.py

*now you can accept http://127.0.0.1:8080/index.html to play game, have fun!*
